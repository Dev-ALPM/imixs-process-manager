Welcome to the Imixs Business Process Manager Suite

Imixs-Workflow is an open source workflow engine based on Jakarta EE. The Imixs-Workflow project provides components to build human-centric workflow applications within a flexible and robust framework. The business logic can be modeled and executed using the Business Process Modelling Notation - BPMN 2.0 standard. See the Project Home for more information. Find out more on www.imixs.org
